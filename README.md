# Worcspace

A responsive React application replicating a Figma-provided Knowledge Base interface, built with functional components, Tailwind CSS, and clean component-based architecture.

---

## Website Link
- https://worcspace.onrender.com/

---

## ✨ Features

* ✅ Pixel-accurate replication of the Figma design
* ✅ Responsive layout across screen sizes
* ✅ Functional "Create New" button that opens a modal pop-up
* ✅ Component-based architecture for scalability and reuse
* ✅ Clean, readable code with clear naming conventions

---

## 🚀 Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/vinaypoddar2000/Worcspace.git
cd worcspace
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

---

## 🛠️ Technologies Used

- **React**: UI framework(Functional Components + Hooks)
- **Tailwind CSS**: Styling and layout
- **JavaScript ES6+**: Core programming logic
